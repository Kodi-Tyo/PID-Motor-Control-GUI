# Description
This repository presents a closed-loop DC motor speed control system driven by a PID controller and encoder feedback. An interactive GUI enables users to monitor motor speed and tune PID parameters in real time. Mechanical mounting designs for the motor, Arduino Uno, and motor driver are also provided.

# Key Feature
- *DC motor speed control* using PID
- *Real-time speed monitoring* with graph via GUI
- *PID tuning* via GUI
- *Communication between Arduino and Python GUI*
